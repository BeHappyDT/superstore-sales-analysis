# Superstore Sales Analysis

## Project Overview

This is a data analytics portfolio project focused on sales and profitability analysis using a sample Superstore retail dataset.

The goal of this project was to explore regional sales performance, category-level revenue, profitability patterns and the impact of discounts using a Power BI dashboard.

The project helped me practice structuring an analysis around business questions, building clear visualizations and translating data into practical insights.

## Dataset

The project uses a sample Superstore retail dataset for educational portfolio purposes.

The dataset includes sales-related information such as:

- regions
- product categories
- sales
- profit
- discounts

## Tools Used

- Power BI
- Data Visualization
- Dashboarding
- Business Analysis
- Git & GitHub

## Business Questions

The analysis was designed to answer the following questions:

- Which regions generate the highest revenue?
- Which categories drive the most sales?
- Are there regions or categories operating at a loss?
- How do discounts impact profitability?

## Key Findings

### 1. Technology generates the highest revenue

The Technology category generated the highest revenue among the analyzed product categories.

### 2. West region leads in sales

The West region showed the highest sales performance, generating approximately 0.73M in sales.

### 3. South region shows the weakest performance

The South region generated the lowest sales among the analyzed regions, with approximately 0.39M in sales.

### 4. Some regions operate at a loss

The analysis showed that some regions or categories may generate sales volume while still underperforming in terms of profitability.

### 5. Discounts negatively impact profit margins

Discounts appear to reduce profitability, especially in lower-performing regions. This suggests that discount strategy should be monitored carefully.
